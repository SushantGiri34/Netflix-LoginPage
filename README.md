# Netflix-LoginPage
# 🎬 Netflix Login Page (Practice) just for begginers

This is a **basic Netflix login page UI clone** created **only for HTML & CSS practice**.  
The project is **not responsive** and does **not include any backend or authentication logic**.

- No JavaScript / No backend
